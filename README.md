### Description.
Simple baremetal LED blink on nRF52840-DK.
Below are steps how to get the tools and setup DK.
To clone the project run:
```bash
$ git clone https://github.com/ShpakovDmitry/nRF52840DK_BareMetal_Blink.git
```
